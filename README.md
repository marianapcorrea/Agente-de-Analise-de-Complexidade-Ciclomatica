# Agente-de-An-lise-de-Complexidade-Ciclom-tica
Implementa um agente capaz de analisar a complexidade ciclomática de funções Python utilizando exclusivamente um script interpretado via Azure Foundry. O agente recebe uma função como texto, executa o cálculo através do script definido e retorna o valor da complexidade, acompanhado de classificação, impactos técnicos e recomendações de melhoria.
